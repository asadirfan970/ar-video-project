Step 1: Use MindAR Image Target Compiler

Go to the official MindAR compiler to create your marker file:

Open MindAR Compiler

Step 2: Upload Your Image

Upload your poster.png file to the compiler. Make sure your image:

  • Has good contrast and clear features
  • Is not too blurry or dark
  • Contains recognizable patterns or text
  • Is in JPG or PNG format

Step 3: Download the .mind file

After compilation, download the generated .mind file and rename it to marker.mind

Step 4: Place in Assets Folder

Move the marker.mind file to your assets/ folder, replacing any existing marker file.

Alternative: Quick Test Marker

For testing purposes, you can download a pre-made marker file:

Download Test Marker

Note: This test marker works with the MindAR example card image, not your poster.png

Step 5: Test Your AR Experience

Once you have the marker file in place:

  1. Open your index.html in a web browser
  2. Allow camera permissions
  3. Point your camera at the target image
  4. Watch the AR video appear!
